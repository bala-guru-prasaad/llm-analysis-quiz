# LLM Analysis Quiz Bot

This is my submission for the TDS LLM Analysis Quiz (29 Nov 2025).

## Features
- Receives quiz URL via POST request
- Uses Selenium headless browser to render JavaScript pages
- Solves the quiz automatically using Google Gemini 1.5-flash (free tier)
- Submits answer to the correct endpoint
- Fully runs on free PythonAnywhere account

## Tech stack (all free)
- Python + Flask
- Selenium + webdriver-manager
- Google Gemini API
- BeautifulSoup
- PythonAnywhere (free tier)
